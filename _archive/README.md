# Project HORIZON Archive

This folder contains retired dashboard release artifacts that were removed from
the active dashboard tree.

- `dashboard-v1.5/` is the archived legacy v1.5 Raspberry Pi app drop.
- `older-rms-packages/` contains superseded RMS installer archives.

Use `dashboard/v1.5.1/dist/tparc-rms-pi-app-2026.06-rev01.16.tar.gz` for the
current Raspberry Pi install package.
