# Documentation

This folder contains project documentation that should travel with the
GitHub repository.

## Files

- `SAFETY_ADVISORY.md` - required safety warnings and non-liability notice.
- `REPOSITORY_STRUCTURE.md` - explanation of the repository layout.
- `STATUS.md` - current bring-up state and verification notes.
- `requirements_matrix.md` - traceability notes against the project prompt.

Read the safety advisory before connecting powered propulsion hardware.
